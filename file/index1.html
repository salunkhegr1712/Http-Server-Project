Theory :

CentOS stands for Community Enterprise Operating System. It is a Community-driven free software which is focused on delivering an open source ecosystem based on Linux distribution. first version of CENTOS It is Launched on 14th May 2004.

Installation Process:

The CENTOS has a easy installation process. You just have to follow step given below and you are ready to use the CENTOS. This steps are written such a begineer can unstand it well, so installation steps are like below ::

1.     Open browser and go to URL https://www.centos.org/download/, this is offical website for downloading image  of CentOS.

2.     Then you have to click on the link under ‘CentOS Linux’ to download the iso file of CENTOS

3.     You have see the first your specification of your device and download the file according to it.

4.     After completion of step 3. It will show you the mirror link for download available for your region. Click on any of the mirror link to download the file.

5.     Then, click on the image file ending with boot iso. Download the file you can also use torrent to download the iso file

6.     As we are done with successfully download you have to create a bootable usb-flash drive in order to install the CENTOS in your system

7.     You find a large number of apps for usb flashing you just have to insert drive and select downloaded iso boot file in that app.

8.     Then wait for completion of flash-able usb-drive

9.     Now Boot/Reboot the PC and enter BIOS mode with help of the f keys which are depends on your device architecture and pc model

10.  After completion of step 9, Now select the your bootable drive from the booting options.

11.  Select the option ‘Install CentOS Linux’ option.

12.  Now, CentOS installation page will appear. Select the prefered language and click continue to go to next step in installation process.

13.   Configure the installation in Installation summary page according to your preference.

14.  There are lot of option are there for setup such like network setup, installation packages, proxy setting, http proxy setting.

15.   Now Select installation destination and select the disk in which CentOS is to be installed. Click done and then, click on Begin Installation key

16.  Now Installation process of CENTOS will begin. While it is completing, then you hav to set the root password and user creation.

17.  Now click on Reboot and remove the flash bootable drive from your system .

18.  System will start booting. If you are doing dual boot then you have select CENTOS from options.

19.  After booting, A page called Initial Setup appear. Noe you have Accept the licence agreement by clicking on Licence Information and check the checkbox below. Click on Finish Configuration.

16.  Then  we completed all installation process now Login screen will appear. Enter your password and enter in CENTOS

17.  With this step CentOS is successfully installed on your PC.
